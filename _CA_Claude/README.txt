ЧТО ЭТО?
--------
Инструмент для автоматического сбора информации о проекте IPAN.
Создает файл context.md с полной структурой проекта для работы с AI (Claude, ChatGPT).


КАК ИСПОЛЬЗОВАТЬ?
-----------------

1. УСТАНОВКА
   - Эта папка (_CA_Claude) должна лежать в корне проекта IPAN
   - Структура должна быть такой:
     
     C:\.code\.ipan\
       ├── _CA_Claude\          <-- эта папка
       │   ├── generate-context.py
       │   └── README.txt
       ├── frontend\
       ├── backend\
       └── ...


2. ЗАПУСК
   
   Откройте командную строку в папке _CA_Claude:
   
   Способ A (в проводнике):
   - Откройте папку C:\.code\.ipan\_CA_Claude
   - Shift + Правая кнопка мыши
   - "Открыть окно PowerShell здесь" или "Открыть командную строку здесь"
   
   Способ B (через cmd):
   cd C:\.code\.ipan\_CA_Claude
   
   Затем запустите:
   python generate-context.py
   
   или:
   python3 generate-context.py
   
   или:
   py generate-context.py


3. РЕЗУЛЬТАТ
   
   Скрипт создаст файл context.md в этой же папке.
   Вы увидите:
   
   _CA_Claude\
     ├── generate-context.py
     ├── context.md           <-- новый файл
     └── README.txt


4. ИСПОЛЬЗОВАНИЕ С AI
   
   - Откройте файл context.md
   - Загрузите его в чат с Claude или ChatGPT
   - Напишите: "Читай context.md и помоги с проектом IPAN"
   
   Теперь AI видит всю структуру вашего проекта!

----------

ПРОБЛЕМЫ?
---------

Проблема: "Python не найден"
Решение: Установите Python с python.org

Проблема: "Файл не создается"
Решение: Убедитесь что запускаете из папки _CA_Claude

Проблема: "Permission denied"
Решение: Запустите от имени администратора
